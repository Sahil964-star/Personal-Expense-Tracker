# Personal Expense Tracker

## Description

A command line based Personal Expense Tracker built using Python.

Users can:

* Add expenses
* View all expenses
* Filter expenses by category

Data is stored in a text file so that expenses remain saved even after closing the program.

## Features

### Add Expense

User enters:

* Amount
* Category
* Note

### View Expenses

Displays:

* Date
* Amount
* Category
* Note
* Total expense

### Filter By Category

Displays all expenses belonging to a selected category along with subtotal.

## Technologies Used

* Python
* Git
* GitHub

## How To Run

Clone the repository:

git clone YOUR_REPOSITORY_LINK

Run:

python main.py
